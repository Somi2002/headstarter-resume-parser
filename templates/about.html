Welcome to our Resume Parser project! This project was developed using Python (Flask) and HTML/CSS. The purpose of this project is to automate the resume filtering process for recruiters. Users can upload their resumes, which are then parsed and filtered for specific keywords. The parsed resume data is stored in the Google Firestore database. Recruiters can use the search bar to enter keywords and filter out the most relevant resumes. This streamlines the candidate selection process, saving time and effort.
